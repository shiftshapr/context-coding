# Context Coding Starter Kit

**New here?** Start with `01_GETTING_STARTED.md` to download/clone this repository.

**Ready to learn?** Read `02_TOOLING_OVERVIEW.md` first for a complete guide to all available tools.

Contents of this directory (read in order):
- `01_GETTING_STARTED.md` – **Start here** - How to get this repo (beginners & experienced users)
- `02_TOOLING_OVERVIEW.md` – **Complete guide to all AI coding tools** (system prompt, Cursor rules, agents, JAUmemory, workflows, Git, ESLint)
- `03_WORKFLOW_GUIDE.md` – **Step-by-step workflow** from task identification to completion (planning, execution, monitoring, troubleshooting)
- `04_SETUP_AND_ONBOARDING.md` – Detailed setup instructions
- `05_SYSTEM_PROMPT.md` – the orchestration system prompt you currently send.
- `06_CURSORRULES.md` – index explaining which rules to use
- `07a_CURSORRULES_GENERIC.md` – universal red-line policies for all projects
- `07b_CURSORRULES_EXTENSION_EXAMPLE.md` – extension-specific rules (browser extensions only)
- `08_DEFAULT_WORKFLOW_MANIFEST.md` – Default Collaboration Workflow Manifest (execution order + reporting requirements).
- `09_JAUMEMORY_CONTEXT_QUICKSTART.md` – practical guide for problem cataloging, diagnostics, blind-spot tracking, and memory consolidation.
- `10_AGENT_DESCRIPTIONS.md` – agent roles/responsibilities reference.
- `11_SYSTEM_DIAGRAM_PROMPT.md` – Visual diagram generation prompt

Share this folder with collaborators so they have the minimum context to execute the workflow end-to-end.
